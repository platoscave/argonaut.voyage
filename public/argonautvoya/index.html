Argonaut.Voyage

Foundation

Brenging process modeling and process execution to the blockchain.

What

Argonaut.Voyage is a process modeling and execution environment that allows users to:

  • Browse other users/organizations content
  • Copy/customize prototype organizations
  • Model processes that support your organization
  • Start using it

Blockchain made easy.

How

Class Model

Argonaut.Voyage is built on a class model called Process Universe. These classes describe all objects needed to model and execute processes. Classes take the form of json schema objects, meaning they can be used for server-side validation and client-side form generation.

Process Model

The process model allows users to define propositions, model processes and sub-processes, model organizations and assign authorizations

Page Model

The page model allows developers to to design master-detail page layouts that follow the data graph (i.e. one to many associations). A page has one or more tabs. Each tab contains one or more widgets or points to a different page. Pages can have a divider, in which case the left/top side serves as a master and the right/bottom side as a detail page. There are several widgets to choose from. Among others: Menu, Table, Tree, Form, several 3D models and financial reports.

Our json schema form component takes a schema as input and presents data in a user friendly form. In edit mode, schema validation rules are applied to user input. Nested objects and arrays of objects are displayed as nested subforms.
